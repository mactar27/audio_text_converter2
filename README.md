# 🎵 Audio Text Converter 2

Une application Flutter qui permet de convertir des fichiers audio en texte automatiquement grâce aux technologies de reconnaissance vocale.

## 📋 Présentation

Audio Text Converter est une application web moderne conçue pour transcrire automatiquement des fichiers audio en texte. Ce projet facilite la transcription d'audios en contenu textuel exploitable, ce qui peut être utile pour les étudiants, les journalistes, ou toute personne ayant besoin de convertir la parole en contenu écrit.

## ✨ Fonctionnalités

- 🎤 **Conversion audio en texte** : Transcription automatique grâce à la reconnaissance vocale
- ⚡ **Traitement rapide** : Conversion efficace des fichiers audio
- 📱 **Interface responsive** : Fonctionne sur tous les appareils (desktop, mobile, tablette)
- 🌐 **Application web** : Accessible directement depuis votre navigateur
- 🎯 **Interface intuitive** : Utilisation simple et conviviale

## 🎯 Objectifs

- Convertir un audio en texte automatiquement
- Gagner du temps sur la transcription manuelle
- Améliorer l'accessibilité des contenus audio
- Fournir un outil simple et efficace

## 🚀 Comment utiliser

1. **Ouvrir l'application** : Accédez à l'application via votre navigateur web
2. **Télécharger un fichier audio** : Cliquez sur le bouton pour sélectionner votre fichier audio
3. **Lancer la conversion** : Démarrez le processus de transcription
4. **Obtenir le texte** : Récupérez le texte transcrit et copiez-le ou téléchargez-le

## 🛠️ Technologies utilisées

- **Flutter** : Framework de développement multiplateforme
- **Dart** : Langage de programmation principal
- **Reconnaissance vocale** : Technologies de conversion parole-texte
- **Font Awesome** : Icônes pour l'interface utilisateur

## 📱 Compatibilité
